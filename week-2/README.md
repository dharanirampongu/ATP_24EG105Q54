# Week 2: Advanced JavaScript

## Overview
Week 2 dives deeper into JavaScript's more advanced features, modular patterns, and asynchronous operations.

## Details
- **Modules**: Using ES6 modules to organize code (`module-demo`).
- **Array Operations**: Advanced array manipulation including spread and rest operators.
- **Asynchronous JS**: Introduction to timer functions (setTimeout, setInterval).
- **OOP Concepts**: Working with classes and object-oriented patterns (`class-demo`).
- **Data Structures**: Deep vs Shallow copying techniques and object destructuring.
- **Server Basics**: Initial server setup and request handling (`server.js`).
