# Week 5: Introduction to React with Vite

## Overview
Week 5 marks the start of Frontend application development using the React library bootstrapped tightly utilizing Vite.

## Details
- **React Fundamentals**: Setting up the project framework components (`Navbar.jsx`, `Footer.jsx`).
- **State and Props**: Working with distinct isolated components such as a `Counter` and dealing with user rendering.
- **Forms and Input Handling**: Creating test forms (`FormDemo.jsx`, `FormAss.jsx`) to practice state management with uncontrolled/controlled inputs (`TestRefTypes.jsx`).
- **API integration**: Building initial component views that consume APIs (`APIDemo.jsx`).
