# Week 4: Introduction to HTML & CSS

## Overview
This week focuses on creating static, styled web pages using core HTML structural elements and styling them effectively with CSS.

## Details
- **HTML Layouts & Elements**: Working on lists (nested and unordered) and structuring multiple tabular datasets (`table1.html`, `table2.html`).
- **Basic Styling**: Adding aesthetic appeal through external CSS templates and stylesheets, focusing on fundamental CSS layout rules.
