# Week 6: Modern React Patterns & Employee Management System

## Overview
Week 6 focuses on advanced React patterns, global state management, and the development of a full-stack Employee Management System.

## Project Plan: Employee Management System
A comprehensive application to manage employee records with secure authentication and CRUD functionality.

### Tech Stack
- **Frontend**: React 19, Tailwind CSS 4, Vite 8.
- **State Management**: Zustand (for global state), React Hook Form (for form handling).
- **Routing**: React Router 7.
- **API Client**: Axios.
- **Backend**: Node.js/Express 5.
- **Database**: MongoDB/Mongoose 9.
- **Security**: JWT (JSON Web Tokens), Bcrypt (password hashing).

### Features
- User Authentication (Login/Register).
- Add, View, Edit, and Delete employee records.
- Responsive UI with Tailwind CSS.
- Global state management for user sessions and employee lists.

## Other Mini-Projects
- **AddingUsers**: Practice with `useState` and elementary form handling.
- **context-app**: Deep dive into the React Context API for thematic/global state.
- **userList**: Dynamic list rendering and component modularization.
