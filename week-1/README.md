# Week 1: JavaScript Basics

## Overview
This week covers the foundational concepts of JavaScript, focusing on syntax, basic data types, and logical operations.

## Details
- **Variables and Data Types**: Understanding how to declare and use variables.
- **Arithmetic Operations**: Performing basic calculations (sum, min, etc.).
- **Conditional Logic**: Using if-else statements to find the greatest among multiple numbers.
- **Functions and Arrays**: Basic function definitions and array summation (`arraysumfunc.js`).
- **Hands-on Assignments**:
  - `Day-1`: Basic math and logic scripts (`sum.js`, `min.js`, `greatestamongtwo.js`).
  - `Day-2`: Employee data management and array functions.
